#Created a REST API using Node.JS and Express integrated with GiHub API.

# Packages installed

-   node = `v14.18.1`
-   express
-   node + typescript
-   swagger-ui-express
-   ts-dotenv
-   joi
-   cors
-   supertest
-   axios
-   eslint-config-prettier

## Installation

installed all module via npm:

```bash
$ npm install
```

## Add .env

```bash
$ cp .example.env .env
```

Add PORT and GITHUB_ACCESS_KEY in .env

Start the server

```bash
$ npm start
```

### Development with tsc --watch

```bash
$ npm run dev
```

## Testing

Testing rest api:

```bash
$ npm test
```
